# Loveable-FoodApp

A modern, colorful, and interactive **food delivery mobile app prototype** designed in [Loveable](https://biteswift.lovable.app/).  
This prototype is mobile-first (iPhone 14 size) and includes clickable interactions, smooth transitions, and realistic app flows.

## Live Demo
Check out the live prototype here: [https://biteswift.lovable.app/](https://biteswift.lovable.app/)

## Features
- User onboarding & signup/login
- Browse restaurants with categories
- Search & filter restaurants by cuisine, price, and rating
- Restaurant menu with images, descriptions, and add-to-cart buttons
- Cart & checkout with price summary, promo codes, and payment options
- Real-time order tracking with map and ETA
- Ratings and reviews system
- User profile with order history, settings, and payment methods
- AI-based recommended dishes
- Loyalty points and favorites

## Prototype Highlights
- Fully clickable mobile prototype
- Smooth screen transitions and animations
- Bottom navigation bar for easy app navigation
- Mobile-friendly layout, spacing, and padding
- Vibrant, food-friendly colors and modern fonts

## How to Use
Open the prototype link and interact with the screens as if it’s a real app. All buttons, cards, and flows are interactive.
